AI Powered Resume Analyzer

An AI-based system that extracts structured information from resumes and predicts job category and specialization using deep learning.

Home | What We Developed | Complete Project | Mission | Vision | Goals | Installation

Abstract

This AI-powered resume analyzer parses resumes, extracts structured information, and evaluates job-fit based on NLP techniques and deep learning. It computes similarity scores between resumes and job descriptions, using semantic and statistical metrics, to enhance recruitment processes.

Method Overview

The system parses unstructured resumes, predicts job titles, extracts skills, and ranks candidates using deep learning.

Key Features

  • Resume Parsing
  • Job Description Matching
  • Title Prediction using ML
  • Semantic Skill Extraction
  • Candidate Ranking & Score Visualization

Information Extracted

  • Job Title
  • Candidate Highlights
  • Direct/Related Skills
  • Experience Match
  • Keyword Score
  • Overall Match Score
