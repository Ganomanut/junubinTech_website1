Welcome to the AI-Powered Resume Analyzer project site.

What We Developed

This project involves a transformer-based resume parser and job matcher built using TensorFlow and NLP.

Complete Project

We integrated resume parsing, job matching, skill extraction, and candidate scoring into one unified system.

Project Mission

Our mission is to streamline the recruitment process through AI-driven resume analysis and intelligent candidate-job alignment.

Project Vision

We envision a recruitment process empowered by AI to minimize bias and maximize efficiency.

Goals

  • Automate resume screening
  • Improve job-candidate matching accuracy
  • Enhance recruiter decision-making

Web App Installation

To run the app locally:

  1. Clone the GitHub repo
  2. Install requirements using pip install -r requirements.txt
  3. Run streamlit run app.py
