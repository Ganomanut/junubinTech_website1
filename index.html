Welcome to AI Powered Resume Analyzer

This web application developed by students of the University of Juba provides advanced AI-powered resume parsing and job matching capabilities.

Use the navigation menu above to explore project details, key features, resources, and demonstrations.

Abstract

The AI Powered Resume Analyzer automates the extraction of key information from resumes and matches candidates with suitable jobs using natural language processing and machine learning techniques.

Method Overview

The project employs state-of-the-art algorithms to parse resumes, extract structured data such as skills, experience, education, and then match these against job descriptions to score relevance.

Key Features

  • Accurate extraction of candidate information from resumes
  • Matching candidates with job descriptions using AI
  • Visualization of matching results and highlights
  • Interactive user interface with multi-page navigation

Information Extracted

The analyzer extracts details including:

  • Personal details and contact information
  • Education history and qualifications
  • Work experience and roles
  • Skills and certifications
  • Resume highlights and summary

Visualizations

Visual reports and graphs are generated to give insights into candidate-job match scores, skill overlaps, and experience timelines.

These visualizations help recruiters quickly assess suitability.
