Installation Guide

To run this AI-powered resume analyzer web app locally:

  1. Clone the GitHub repository: git clone https://github.com/Ganomanut/Updated-AI-Powered-Resume-Analyzer
  2. Ensure Python 3.x and required packages are installed (check requirements.txt)
  3. Run the backend server: python app.py
  4. Open index.html in your browser to use the frontend

Make sure to update paths to model files and datasets as needed, and configure your local web server if using one like XAMPP.
