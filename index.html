AI Powered Resume Analyzer

An AI-based system that extracts structured information from resumes and predicts job category and specialization using deep learning.

Abstract

This AI-powered resume analyzer parses resumes, extracts structured information, and evaluates job-fit using NLP techniques and deep learning.

Method Overview

The system parses resumes, predicts job titles, extracts skills, and ranks candidates using AI.

Key Features

  • Resume Parsing
  • Job Description Matching
  • Title Prediction using ML
  • Semantic Skill Extraction
  • Candidate Ranking & Score Visualization

Information Extracted

  • Job Title
  • Candidate Highlights
  • Direct/Related Skills
  • Experience Match
  • Keyword Score
  • Overall Match Score
