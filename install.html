Web App Installation Guide

Follow these steps to install and run the AI-Powered Resume Analyzer locally:

  1. Install XAMPP or another Apache + MySQL environment.
  2. Clone or download the project from the GitHub repository.
  3. Place the project folder inside the htdocs directory of XAMPP.
  4. Start Apache and MySQL servers using the XAMPP control panel.
  5. Import the provided database SQL file into MySQL (if applicable).
  6. Access the web app by navigating to http://localhost/your-project-folder/ in your browser.

For more details or troubleshooting, check the README file in the GitHub repo.
