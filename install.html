Web App Installation

To run the AI-Powered Resume Analyzer locally:

1. Clone the repo:
   git clone https://github.com/Ganomanut/Updated-AI-Powered-Resume-Analyzer

2. Navigate to project:
   cd Updated-AI-Powered-Resume-Analyzer

3. Install dependencies:
   pip install -r requirements.txt

4. Run the app:
   streamlit run app.py

You can also try the Google Colab version.
