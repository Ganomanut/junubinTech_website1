Complete Project Details

This project combines AI, machine learning, and natural language processing to create an intelligent resume analyzer. It supports:

  • Parsing resumes to extract key data
  • Matching resumes against job descriptions
  • Predicting job titles from candidate data
  • Visualizing match scores and highlights
  • Integrating with existing HR platforms

The system uses TensorFlow for model training, Google Drive for data storage, and a clean web interface for user interaction.
